# UI-Aplikasi-Daily-Task-Management
Aplikasi Daily Task Management ini fokus pada kesederhanaan (simplicity) dan kemudahan navigasi (user-friendly). Fokus utama dari desain ini adalah membantu pengguna mengorganisasi prioritas harian, melacak progres, dan membangun kebiasaan produktif melalui antarmuka yang menyegarkan secara visual.
